                        JAVADOC2HTML
Версия 1.0
Автор: Катяева Дарья

Описание:   Программа получает на вход папку с java-файлами(проектом)
	        и создает документацию для java-файлов в формате html.


        +парсинг и конвертирование в html:
		    -классов
		    -интерфейсов
		    -полей
            -методов
		    -комментариев
		    -импортируемых модулей и пакетов
	    +создание css файла
        +тесты



Пример запуска:      python javadoc2html.py (если без аргументов, то по умолчанию пaпка тест)
                     python javadoc2html.py -p test
